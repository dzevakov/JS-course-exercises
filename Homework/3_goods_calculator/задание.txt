1) Поддержка разных пользователей
2) Есть список товаров разного типа.
- Обычный товар, картинка + описание + цена
- Экотовары - зеленая рамка, картинка + описание + цена
- Товары без фото - нет картинки, только описание и цена
- Товары со скидкой - Картинка + описание, цена с учетом коэф скидки

3) В коде прописать по паре товаров каждого типа.
4) Дать возможность пользователю, выбирать товары из списка в п3.
Выводить ему общую сумму выбраных товаров.
5) Дать возможность пользователю сохранять этот список и итоговую сумму.
6) Очевидно что если входит второй пользователь, то он видит начальный экран
или СВОЙ список товаров которые он выбрал в прошлый раз.


Примечания
-----------------------------
- Разбить задачу на маленькие части которые понятно как делать.
- Простой html с минимальными стилями вроде рамок.

- Сохранение делать через localStorage и JSON.stringify JSON.parce
- Для работы с пользователями сделать CRUD методы (Create, Read, Update, Delete)
- Работа с сервером здесь не нужна.

- Продумать как лучше сделать типы товаров через классы.
- Сделать аккуратную структуру проекта и кода.
